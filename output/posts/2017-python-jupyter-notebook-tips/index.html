matplotlib 그래프를 결과창에 출력¶

matplotlib를 사용하여 만들어진 다른 유용한 그래프 라이브러리를 사용시에도 필요합니다.

  • Seaborn
  • mpld3
  • bokeh
  • plot.ly
  • Altair
In [6]:
import matplotlib.pyplot as plt
%matplotlib inline

plt.plot(df); # 출력시 ; 를 사용하면 결과창에 그래프만 출력

단축키¶

  • Esc: 명령 모드로 전환
    • a : 현재 셀 위에 새로운 셀 추가
    • b : 현재 셀 아래 새로운 셀 추가
    • dd : 현재 셀 삭제
    • y : 현재 셀을 코드 모드로 전환
    • m : 현재 셀을 마크다운 모드로 전환
  • Enter : 현재 셀을 편집 모드로 전환
  • Ctrl + Shift + - : 셀 상의 현재 커서 위치에서 셀을 2개로 나눔
  • Esc + f : 셀안에서 문자 찾아 바꾸기
  • Esc + o : 셀 결과창 나타내기/숨기기
  • Shift + 커서(Up/Down) : 여러 셀 선택
  • Shift + m : 선택된 여러샐 병합
  • ?? -> Shift + Tab : 함수의 설명 보기 (예: str.replace?? 후에 Shift + Tab)
