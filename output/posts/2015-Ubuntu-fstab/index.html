2. 부팅 후 하드디스크 확인¶

5. 마운트¶

uuid 확인

> sudo blkid
...
/dev/sdb1: UUID="d2eac947-9407-40bb-96cb-acd22e9a57c3" TYPE="ext4"
...

fstab 파일에 파티션 추가

> sudo vi /etc/fstab
UUID=d2eac947-9407-40bb-96cb-acd22e9a57c3 /ssd2 ext4 defaults 0 0

마운트 및 확인

> sudo mkdir /ssd2
> sudo mount -a
> df -h

...
