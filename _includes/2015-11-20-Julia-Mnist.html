MNIST Datasets in Julia¶

MNIST Dataset은 0부터 9까지의 필기체를 모아 놓은 데이터 셋으로 6만개의 train set 과 1만개의 test set으로 이루어져 있으며

Machine Learning에서 자주 사용되는 데이터 셋입니다.

http://yann.lecun.com/exdb/mnist/

여기서는 Julia 에서 해당 데이터셋을 MNIST 패키지를 통해서 읽어와서 그중 하나의 이미지를 PyPlot 패키지를 통해 plotting 해보겠습니다.
